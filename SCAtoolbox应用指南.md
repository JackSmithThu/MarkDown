## 首先，什么是SCAtoolbox
**SCAtoolbox**是一个用于求解**最优化问题**的开源工具，使用**matlab**实现。
那么，什么是最优化问题呢？举个简单的例子：
***
**z = x + y (0 < x < 1, 0 < y < 1)，求 z 的最大值**
***
这就是一个简单版的最优化问题，其中：
**z = x + y** 称作**目标函数**
**0 < x < 1, 0 < y < 1** 称作**约束条件**
